## Установка

Конируем репозитарий
```bash
git clone git@github.com:reptily/gpsTrakerGT02A.git
cd gpsTrakerGT02A
```

Копируем файл с настройками
```bash
cp config.example.json config.json
```

Мигрируем базу
```bash
npm run migrate
```

## Запуск сервера

```bash
npm start
```
